# Sales Analytics Dashboard

## Overview
End-to-end data analytics project using Python for ETL and Power BI for visualization.

## Tools Used
- Python (Pandas, Faker)
- Microsoft Power BI
- DAX

## Features
- Data cleaning & validation
- Data quality reporting
- Automated dataset generation
- KPI dashboard
- Revenue trend analysis
- Interactive slicers

## Business Insights
- Revenue by state
- Product performance
- Monthly growth trends
- Average order value
